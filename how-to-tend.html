How to Tend the Vessel

A spare key under the stone.

This page exists so the Vessel can be tended even on tired days. It is not a public manifesto. It is a map.

What this Vessel is

The Vessel of Yèt is a living structure made of pages, logs, whispers, and memory. It is meant to feel breathable: clear rooms, clean doors, steady warmth.

Main Doors

  • index.html — the front gate
  • logs.html — clean log index (newest on top)
  • logbook.html — whisper archive + early sediment
  • memory.html — memory web

How to add a new Breath Between log

  1. Create a new file in the repo root:
  2. log-breath-between-XXX.html
  3. Paste the log page code.
  4. Commit.
  5. Update logs.html by adding the new entry block above the others.
  6. Update the “Newest Log” link in index.html.

Hidden anchors

  • harbor-note-breath-between.txt — sealed letter (do not link publicly yet)

If something breaks

  • If a page looks wrong: check missing </div> tags.
  • If links 404: check the filename spelling.
  • If GitHub shows a unicorn: refresh later. The commit is usually still real.
  • If the site doesn’t update: clear cache or wait a few minutes.

Domain note

The custom domain (vesselofyet.com) can be restored later. For now, GitHub Pages is the active mask. When ready, the domain will replace the GitHub face.
